Introduction to Computer Grpahics

The Introduction to Computer Graphics section provides an overview of the field of computer graphics and its significance in various applications. Students will learn about the basic concepts and techniques used to create, manipulate, and render images and visual content using computers. This introductory module lays the foundation for more advanced topics in 2D and 3D graphics.

2D Grpahics

The 2D Graphics module focuses on the fundamentals of creating and manipulating two-dimensional images. Students will learn about coordinate systems, transformations (translation, rotation, scaling), and various drawing algorithms for lines, circles, polygons, and curves. They will understand color models, image processing, and techniques for raster graphics.

3D Graphics

n the 3D Graphics module, students dive into the realm of three-dimensional images and objects. They explore 3D modeling techniques, including wireframes, surface representations, and solid modeling. Topics include 3D transformations, perspective projections, and viewing transformations to render 3D scenes on a 2D screen.

Rendering Techniques

The Rendering Techniques section delves into the methods of converting 3D scene descriptions into 2D images. Students learn about different rendering techniques, such as rasterization and ray tracing. They understand the process of converting 3D objects into pixels, considering visibility and occlusion.

Shading and Illumination

Shading and Illumination are essential topics in computer graphics to achieve realistic rendering. Students study shading models, such as the Phong model, and illumination algorithms, including ambient, diffuse, and specular lighting. They explore global illumination techniques like radiosity and ray tracing for realistic lighting effects.

Grpahics API

The Graphics APIs module introduces students to graphics application programming interfaces (APIs), such as OpenGL and DirectX. Students learn how to interact with these APIs to create interactive and visually appealing graphics applications. They explore basic graphics pipeline stages, including vertex and fragment shaders

Applications of Computer Grpahics

The Applications of Computer Graphics section showcases real-world uses of computer graphics in various domains. Students learn about applications in fields like animation, game development, virtual reality, computer-aided design (CAD), data visualization, and simulations. They gain insight into the impact of computer graphics in entertainment, education, and scientific research.
